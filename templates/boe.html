BOE Bots

10/2007-12/2007

Description

First foray into MCU-based electronics. The challenge for the robot involved an autonomous traversal of an obstacle course. To meet it, we used a number of off-the-shelf components, including a Parallax development board, QTI line follower, infrared LED/detector pairs, servomotors, and an RFID transmitter/receiver pair.

The software operated in seven separate memory banks on the BS2sx microcontroller. Although the program used all available slots, only about 5000 bits were utilized in completing the challenge (discounting all common methods i.e. servo controller methods). Organizing each step of the challenge into separate memory banks simplified analysis and debugging.
